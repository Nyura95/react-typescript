# react-redux-typescript

Typescript 3.4.5

## Overview

Here is my vision of a framework react (typescript v3) with redux. 

## Installation

> `git clone https://github.com/Nyura95/react-typescript.git`

> `cd ./react-typescript`

> `npm install`

## Development mode

> `npm run start`

## build

> `npm run build`

## run a server in production

> `npm run serve`

## lint your code

> `npm run tslint`

Enjoy !
