# react-redux-typescript

Typescript 3.4.5
Redux v7.1.0

## Installation

> `git clone https://github.com/Nyura95/react-typescript.git`

> `cd ./react-typescript`

> `npm install`

## Development mode

> `npm run dev`

## Development mode with hmr and cache

> `npm run start`

## build

> `npm run build`

## run a server in production

> `npm run serve`

## lint your code

> `npm run tslint`

Enjoy !
